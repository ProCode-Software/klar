# Klar Repository Code Style Guide

With AI writing more code in large codebases, it is important to have a consistent code style across the project. We want high-quality, educational, and maintainable code in this project. AI doesn't tend to write code that is maintainable for years, and writes sloppy code that no skilled human would write. So we have created some style guides for humans and LLMs to follow to keep the Klar codebase clean and consistent.

- [Go](./GoStyleGuide.md)
